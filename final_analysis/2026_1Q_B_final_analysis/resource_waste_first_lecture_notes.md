# 第 1 回 `資源と廃棄物` 補足ノート

このノートは、最初に提供された「第 1 回講義資料メモ 最終ページのまとめ」を出発点とし、その後 資源・廃棄物系講義資料メモ の原講義資料で確認して整理したものである。past exam の代替ではなく、`Q1` の出題範囲を補強するための資料である。

## なぜ重要か

歴年 B クラスの `Q1` は `MR / CR / TR` を安定して問う。しかし、この講義資料からは、教員が第 1 回で実際に扱った範囲がより広いことが分かる。

- 廃棄物量と最終処分場の圧迫
- プラスチック廃棄物の地球規模の蓄積と難分解性
- マイクロプラスチック / ナノプラスチックのリスク
- 3R
- バイオプラスチック
- `CO2` 資源化、`CCS / CCUS`
- `バーゼル条約`
- 大学内の化学廃棄物管理、`PRTR`、`IASO`

したがって、`Q1` は「リサイクル定義問題」から「現状 + 危害 + 対策」へ広がる可能性がある。

## この補足が影響しやすい問題

### 最も直接: Q1 資源と廃棄物

解答に入れられる要点は次のとおりである。

- 一般廃棄物の負担と最終処分場の有限性
- プラスチックの難分解性と地球規模の拡散
- マイクロプラスチックの食物連鎖への取り込み
- ナノプラスチックが生体バリアを通過する可能性
- `3R`
- `MR / CR / TR`
- `バイオマスプラスチック`
- `生分解性プラスチック`
- `CCUS / カーボンリサイクル`
- `バーゼル条約` と `汚れたプラスチックごみ` の越境移動規制

### 次に直接: Q6 化学物質 / 生命環境系

この講義の次の内容は `Q6` 側へ移る可能性がある。

- `PRTR`
- 実験廃棄物の分類と管理
- 法規制の遵守
- `IASO`

### 間接: Q4 持続可能社会の総合論述

持続可能社会の課題として `資源循環`、`廃棄物`、`プラスチック汚染` を選ぶ場合、このノートの内容をそのまま解答の骨格にできる。

## 暗記方法

数値をすべて厳密に暗記するより、次の種類の情報を解答に書けるようにする。

1. 問題の規模
   廃棄物量が大きいこと、最終処分場が有限であること、プラスチックが長期残留すること。

2. リスクの仕組み
   マイクロプラスチックが食物連鎖に入り、ナノプラスチックが体内へ入る可能性があること。

3. 基本対策
   `3R` と `MR / CR / TR`。

4. 発展的対策
   `バイオプラスチック`、`生分解性プラスチック`、`CCUS / カーボンリサイクル`。

5. 管理制度  
   `バーゼル条約`、`PRTR`、大学内化学廃棄物管理、`IASO`。

## 講義資料メモ から新たに確認した高価値ポイント

1. `バーゼル条約`
   - 講義資料では、有害廃棄物の越境移動に関する国際規制として扱われている。
   - `2019` 年改正で `汚れたプラスチックごみ` が対象に入った。
   - 過去の正誤問題における「パリ協定ではなくバーゼル条約」という論点と直接対応する。

2. `3R` の制度背景
   - 講義資料では、日本が `2000年 循環型社会形成推進基本法` で `3R` を導入したことが明記されている。

3. 日本で `サーマルリサイクル` が多くなりやすい理由
   - 講義資料では、日本の焼却技術が発達していること、分別・再資源化には手間と人件費がかかること、過去には大量輸出があったことが示されている。
   - 中国や東南アジアが輸入を制限した後、国内処理の問題はより明確になった。

4. `プラスチック問題の三大キーワード`
   - `高い安定性`
   - `親油性`
   - `添加剤`

   これらは A クラス過去問でも反復して出ているが、もともとは第 1 回講義資料にも含まれている。
