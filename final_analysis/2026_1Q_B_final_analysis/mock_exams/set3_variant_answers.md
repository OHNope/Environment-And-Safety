# 2026 1Q 環境安全論 B 模擬試験 Set 3 解答

## 解答例の要点

### 問1

- プラスチックは分解されにくく、環境中に長期残留する。
- 親油性により有害化学物質を吸着しやすく、添加剤も問題となる。
- マイクロプラスチックは食物連鎖に入り、ナノプラスチックは生体内へ移行するおそれがある。
- `3R`: Reduce、Reuse、Recycle。
- `MR`: 材料として再利用、`CR`: 化学原料へ戻す、`TR`: 熱回収。
- 対策として、分別・洗浄による MR / CR の高度化、バイオマスプラスチック、生分解性プラスチック、CCUS、カーボンリサイクル、制度的管理を組み合わせる。

### 問2

- 科学性: 調査、予測、評価をデータと専門知に基づいて行うこと。
- 民主性: 情報公開、住民参加、意見聴取を通じて社会的受容性を反映すること。
- 再生可能エネルギーは脱炭素に有効だが、森林伐採、景観、生態系、騒音、土砂災害、住民対立を生むことがある。
- 科学的評価だけでなく、地域住民との合意形成が不可欠である。

### 問3

- `エクセルギー` は、エネルギーのうち有用な仕事として取り出せる部分である。
- 電気エネルギーは質が高く、低温熱は質が低い。
- `800℃ = 1073 K`、`25℃ = 298 K`。
- `η = 1 - 298 / 1073 ≈ 0.722`、約 `72.2%`。

### 問4

どのテーマを選んでも、次の構成で書く。

- 課題の内容。
- 原因。
- 環境・安全・社会への影響。
- 技術的対策。
- 制度的・社会的対策。

資源循環を選ぶ場合は、廃棄物削減、リサイクル、材料設計、回収制度をつなげるとよい。

### 問5

- `DT` 計算では、反応前後の質量差を質量欠損として求め、`E = Δmc^2` でエネルギーへ換算する。
- `eV` への換算を行い、標準値は約 `1.8×10^7 eV`。
- `MPI` は多次元貧困指数であり、所得だけでなく教育、健康、生活水準など複数の欠乏を評価する。
- エネルギー貧困は、照明、調理、冷暖房、教育、医療、情報アクセスに影響し、持続可能社会の基盤を損なう。

### 問6

- 生物多様性の喪失は、生物資源、食料、医薬品候補、生態系サービス、文化的価値を失わせる。
- `キーストーン種` は個体数が少なくても生態系全体へ大きな影響を持つ種である。
- 例としてラッコを挙げると、ウニを抑制し、海藻林を維持する。
- バイオマスプラスチックは、原料がバイオマス由来のプラスチック。
- 生分解性プラスチックは、微生物などにより分解される性質を持つプラスチック。
- バイオマス由来でも生分解性とは限らず、生分解性でも化石資源由来の場合がある。

### 問7
