# A クラス 暗記用ミニマム版

- Q2: ティッピングポイント、`緩和 / 適応`、`Our Common Future 1987`、`MDGs 8`、`SDGs 17`

- Q4: 電気は質が高い、低温熱は質が低い、`エクセルギー`

## 1. 廃プラスチック

プラスチックは `安定性` が高く分解されにくいため、環境中に長く残り、マイクロプラスチックやナノプラスチックになる。

`親油性` により疎水性汚染物質を吸着し、生物摂取や食物連鎖を通じて移動するおそれがある。

`添加剤` として可塑剤、難燃剤、安定剤などを含み、溶出すると毒性や内分泌かく乱が懸念される。

## 2. MR / CR / TR

- `MR`: マテリアルリサイクル。材料として再利用。
- `CR`: ケミカルリサイクル。分解して化学原料へ戻す。
- `TR`: サーマルリサイクル。焼却・燃料化で熱回収。
- `MR / CR` には、単一素材に近く、汚れ・異物・劣化・添加剤混入の少ないプラスチックが必要。

2026 quiz 数字:

- 3 種合計利用率 `85%`
- TR 除外 `25%`
- 廃プラスチック `9.4百万トン`
- 輸出量 `168万トン`
- 輸出先 `中国`
- 条約 `バーゼル条約`
- 2019 年追加対象 `廃プラスチック`

長答の型: 現状 → 危害 → `3R / MR / CR / TR` → バイオマスプラスチック・生分解性プラスチック・CCUS → `バーゼル条約`。

## 3. カーボンニュートラル

科学技術の貢献: 省エネ、再エネ、蓄電池、水素、燃料電池、`CCS / CCUS`、カーボンリサイクル、長寿命材料、リサイクル高度化。

注意: 使用時だけでなく、原料採取から廃棄までのライフサイクル全体で排出削減を評価する。

## 4. ティッピングポイント

ある閾値を超えると、環境システムが急激・非線形・不可逆的に別状態へ移る点。

例: 永久凍土融解、氷床融解とアルベド低下、サンゴ礁白化、森林枯死。

## 5. 持続可能性

- `緩和`: 温室効果ガス排出を減らす。省エネ、再エネ、電化、森林保全、CCS。
- `適応`: すでに起きる影響に備える。洪水、熱中症、防災、農作物対策。
- `Our Common Future`: 1987 年。
- `MDGs`: 2000 年、8 目標。
- `SDGs`: 2015 年、17 目標、2030 年目標、全世界対象。
- CO2 排出を減らしても気温はすぐ戻らない。海洋熱蓄積、長寿命 CO2、気候システムの遅れがある。
- 法律や政策は科学だけで決まらず、政治的・社会的判断も含む。
- `MPI`: 多次元貧困指数。0-1、大きいほど貧困が深刻。

## 6. 放射線

- 防護三原則: `時間 / 距離 / 遮へい`
- `α線`: ヘリウム原子核
- `β線`: 電子
- `γ線 / X線`: 電磁波
- `中性子線`: 水やポリエチレンなど水素を多く含む物質で弱めやすい
- `確定的影響`: しきい線量あり。重症度が増す。例: 脱毛、白内障、皮膚障害、不妊
- `確率的影響`: 発生確率が増す。例: がん、白血病、遺伝的影響
- 世界平均自然放射線量: `2.4 mSv/年`

## 7. エネルギー

電気エネルギーは仕事へ変換しやすく質が高い。低温熱は仕事へ変換しにくく質が低い。この違いは利用可能な仕事量である `エクセルギー` で説明する。

太陽光発電は熱機関ではないので、同じ意味でカルノー効率制限を受けない。

- `CCS`: CO2 を分離・回収・貯留
- `CCUS`: 回収 CO2 を利用もする
- 水素: 多様なエネルギー源から製造でき、貯蔵・運搬できる
- 燃料電池: 水素と酸素から直接電気を取り出す。白金触媒とコストが課題
- `3E+S`: Energy Security、Economic Efficiency、Environment、Safety

## 8. グリーンケミストリー

グリーンケミストリーは、有害物質の使用・生成を減らし、汚染を発生源で防ぐ考え方。第 1 原則は廃棄物を出さないこと。

サステイナブルケミストリーは、ライフサイクル、資源循環、再生可能資源、エネルギー効率、社会・経済面まで見る。

- `CFC`: 安定・不燃・低毒性で便利だったが、オゾン層破壊が問題化
- `PFAS / PFOS`: 難分解性で長期残留・広域拡散
- 共通点: 便利で広く使われたが、後から地球規模問題が判明
- `Novel entities`: 新規化学物質などによる未知リスク
- `CAS REGISTRY`: 約 `300,000,000` 種
- 工業化学物質: 約 `40,000-60,000` 種

## 9. PRTR / SDS

- `PRTR`: 事業者が対象化学物質の排出量・移動量を届け出て、国が集計・公表する制度。
- `SDS`: 危険有害性、取扱い、保管、応急措置などを伝える安全データシート制度。
- `PRTR = 515種類`
- 排出量上位代表: `トルエン`

## 10. 生命環境・バイオマス

- `エコロジカル・フットプリント`: 人間活動を支えるために必要な生態系需要量。
- `キーストーン種`: 個体数は少なくても生態系への影響が大きい種。例: ラッコ。
- `フラッグシップ種`: 保全活動の象徴。2026 quiz では `トキ`。
- `IARC Group 1`: 人に対して発がん性あり。アルコール飲料、ヒ素、アスベスト、放射性ヨウ素。コーヒー飲料は違う。
- がんの進展: `発がん → 血管の獲得 → がんの転移`
- バイオマスは短期炭素循環なのでカーボンニュートラルとされる。ただし栽培・輸送・加工の排出は別。
- バイオリファイナリー原料: `生ゴミ / 活性汚泥 / 稲わら / 廃てんぷら油`
- バイオマスプラスチックは炭素源がバイオマス由来。
- 生分解性プラスチックは微生物などで分解される。両者は同じではない。

## 11. 核融合

- `E = mc^2`
- `DT` 核融合: 重水素 `D` + 三重水素 `T` → ヘリウム + 中性子 + エネルギー
- 放出エネルギー: `17.6 MeV = 1.76×10^7 eV`
- トリチウムは放射性水素同位体。内部被ばく管理が重要。
- リチウムはトリチウム増殖に重要。
- 1 GW 核融合炉のリチウム消費量: `100-150 kg/年`
- 核融合は連鎖反応ではなく暴走しにくいが、中性子、放射化、トリチウム管理は必要。

## 12. 計算

```text
100 g 質量欠損:
E = 0.1 × (3.0×10^8)^2 = 9.0×10^15 J

太陽:
1.989×10^30 × 0.001 = 1.989×10^27 kg
1.989×10^27 / 4.2×10^9 = 4.7357×10^17 s
4.7357×10^17 / (3600×24×365) ≈ 1.5×10^10 年 = 150億年

東京ドーム:
Q = 1.24×10^9 × 4200 × 80 = 4.1664×10^14 J
9.0×10^15 / 4.1664×10^14 ≈ 21.6 個

1 GW 核融合炉:
10^9 × 3.1536×10^7 = 3.1536×10^16 J
3.1536×10^16 / 3.58×10^-12 ≈ 8.8×10^27 個
(8.8×10^27 / 6.02×10^23) × 7 g ≈ 102 kg/年

海水中リチウム:
14×10^17 × 1000 × 0.17 mg/L = 2.38×10^14 kg
1000基 × 102 kg/年 = 1.02×10^5 kg/年
2.38×10^14 / 1.02×10^5 ≈ 2.3×10^9 年 = 23億年
```

## 13. 個人の省エネ・省資源

例: 照明を消す、冷暖房設定を控える、公共交通を使う、マイボトル、食品ロス削減、分別。

書き方: その行動が電力消費、CO2、資源消費、廃棄物のどれを減らすかを書く。

## 14. 正誤

1. 廃棄物処理法で産業廃棄物の越境移動が規制された。  
   `誤`。`バーゼル条約`。
2. 太陽光発電は熱機関ではないのでカルノー効率を受けない。  
   `正`。
3. 人工化学物質の影響は容易に予測できる。  
   `誤`。
4. 化学合成乳酸由来のポリ乳酸はバイオマスプラスチックである。  
   `誤`。炭素源を見る。
5. CO2 排出量を 0 にすれば温暖化はすぐ戻る。  
   `誤`。
6. 世界平均自然放射線量は 240 mSv/年。  
   `誤`。`2.4 mSv/年`。
7. SDGs は 2000 年に作成された。  
   `誤`。2000 年は MDGs。SDGs は 2015 年。
8. SDGs は 17 目標、MDGs は 8 目標。  
   `正`。
9. 持続可能性は SDGs とともに生まれた。  
   `誤`。
10. 法律や条例は必ず科学だけに基づく。  
   `誤`。
11. PFAS / PFOS は難分解性のため問題になる。  
   `正`。
12. 菅政権では温室効果ガス削減目標が変わらなかった。  
   `誤`。2030 年度 `46%削減` へ引き上げ。
