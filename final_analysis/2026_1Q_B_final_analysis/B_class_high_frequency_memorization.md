## 1. `MR / CR / TR` とは何か。`MR / CR` にはどのようなプラスチックが必要か

`MR` は、廃プラスチックを粉砕、洗浄、溶融、成形などの物理的処理によって再び材料として利用する方法である。`CR` は、熱分解、ガス化、モノマー化などによって化学原料へ戻す方法である。`TR` は、焼却して熱エネルギーを回収する方法である。`MR` と `CR` には、`TR` よりも単一素材に近く、汚れ・異物・劣化・添加剤混入の少ないプラスチックが必要である。

2026 年 B 第 1 回 quiz の穴埋めでは、日本の 3 種合計の有効利用率は `85%`、`TR` を除くと `25%` とされた。また、廃プラスチック `9.4百万トン` のうち `168万トン` が輸出され、最大輸出先は `中国` であった。国境を越える廃棄物移動規制は `バーゼル条約` であり、2019 年に `廃プラスチック` が対象に追加された。

## 1.5 第1回 `資源と廃棄物` で一緒に覚えるべき事項

この回はリサイクルの定義だけでは終わらない。`Q1` に混ざる可能性があるのは次の内容である。

1. `一般廃棄物` と最終処分場  
   1 人 1 日あたり約 `1kg` のごみを出しており、最終処分場の残余年数も限られている。廃棄物問題は処分容量の問題でもある。

2. `プラスチック廃棄物` の深刻さ  
   プラスチックは分解されにくく、環境中に長期残留する。

3. `マイクロプラスチック / ナノプラスチック`  
   マイクロプラスチックは食物連鎖に入り、ナノプラスチックは生体内に取り込まれて蓄積するおそれがある。

4. `3R` を超えた対策  
   - `バイオマスプラスチック`
   - `生分解性プラスチック`
   - `CCS / CCUS`
   - `カーボンリサイクル`

5. `バーゼル条約`  
   有害廃棄物や「汚れたプラスチックごみ」の国境を越える移動を規制する枠組みであり、`パリ協定` ではない。

6. カーボンニュートラルへの科学の貢献  
   省エネルギー、再生可能エネルギー、蓄電、水素、CCS / CCUS、カーボンリサイクル、材料の長寿命化、リサイクル高度化を組み合わせ、ライフサイクル全体で排出を減らす。

## 2. 環境アセスメントにおける `科学性` と `民主性`

科学性とは、調査・予測・評価を行い、データや専門知に基づいて環境影響を回避・低減する考え方である。民主性とは、情報公開、住民参加、意見聴取、コミュニケーションを通して、地域の価値観や社会的受容性を反映させる考え方である。

## 3. なぜ再生可能エネルギー事業も環境アセスメントの対象になったのか

再生可能エネルギーは脱炭素に有効だが、導入拡大により森林伐採、土地改変、土砂災害、景観破壊、生態系影響、騒音、住民との対立などが顕在化したためである。課題は、脱炭素の推進と地域環境保全、住民参加、制度の迅速性をどう両立するかにある。

## 4. エネルギーの形態と質の違い

例として電気エネルギーと熱エネルギーを挙げる。電気エネルギーは仕事へ変換しやすく、質が高い。熱エネルギーは、とくに低温熱ほど仕事へ変換しにくく、質が低い。この質の違いは、利用可能な仕事量、すなわち `エクセルギー` の違いである。

## 5. 熱機関の最大効率

`カルノー効率` は `η = 1 - Tc / Th` で与えられる。温度は絶対温度で扱う。室温 `25℃`、高温部 `1000℃` なら、`Tc = 298 K`、`Th = 1273 K` なので、`η ≈ 1 - 298/1273 ≈ 0.766`、すなわち約 `76.6%` である。

## 6. `DT` 核融合 1 回で放出されるエネルギー

標準解答は `1.8×10^7 eV` である。解答では、質量欠損 `Δm = 0.020 mp` を書き、`E = Δmc^2` を用いてから `eV` に換算するとよい。

```
D + T → He + n + 17.6MeV
```

## 7. 原子力発電と核融合発電の安全性の違い

原子力発電は核分裂の連鎖反応を利用し、停止後も崩壊熱が残るため、冷却と長寿命放射性廃棄物の管理が重要になる。核融合発電は高温高密度条件が崩れると反応が止まるため暴走しにくいが、トリチウム管理、中性子照射、材料放射化などの課題がある。

## 8. 持続可能な社会に関する論述の書き方

空論ではなく、次の 4 段で書くと安定する。

1. 問題を 1 つ挙げる  
   例：気候変動、資源枯渇、廃棄物、生物多様性の損失

2. 原因を書く

3. 技術的対策を書く  
   例：省エネ、再エネ、電化、循環利用

4. 制度・社会的対策を書く  
   例：規制、価格付け、インフラ整備、住民合意

### 解答例 1：気候変動

持続可能な社会を実現する上で、重要な問題の一つは気候変動である。化石燃料の大量消費により `CO2` などの温室効果ガスが増加し、地球温暖化、異常気象、海面上昇、生態系への影響が生じるおそれがある。したがって、エネルギー利用の効率を高める省エネルギー、太陽光・風力などの再生可能エネルギーの導入、電化、蓄電池や水素の利用、`CCS / CCUS` などの技術を組み合わせて排出量を減らす必要がある。また、技術だけでなく、炭素税や排出量取引などの価格付け、環境規制、送電網や充電設備などのインフラ整備、地域住民との合意形成も必要である。このように、技術的対策と社会制度を組み合わせ、生活や産業の仕組み全体を低炭素型へ変えることが持続可能な社会につながる。

### 解答例 2：廃棄物と資源循環

持続可能な社会を考える上で、廃棄物と資源枯渇の問題も重要である。大量生産・大量消費・大量廃棄の社会では、限りある資源を消費し続けるだけでなく、最終処分場の不足、海洋プラスチック、マイクロプラスチックなどの環境問題を引き起こす。
特にプラスチックは安定で分解されにくく、環境中に長く残留し、生物に取り込まれるおそれがある。

対策としては、まず `Reduce`、`Reuse`、`Recycle` の `3R` を進め、廃プラスチックについては状態に応じて `MR`、`CR`、`TR` を適切に使い分ける必要がある。

さらに、バイオマスプラスチック、生分解性プラスチック、カーボンリサイクルなどの技術も活用できる。

一方で、分別回収制度、リサイクルしやすい製品設計、企業への規制や情報公開、国境を越える廃棄物移動を管理する `バーゼル条約` のような制度も不可欠である。

資源を一方的に使って捨てる社会から、循環的に利用する社会へ変えることが持続可能性を高める。

## 9. 元素戦略とは何か

元素戦略とは、有害元素や希少元素への依存を減らし、豊富で安全な元素を用いて**同等以上**の性能を実現する材料設計・材料開発の考え方である。

代替対象は `RoHS` 規制物質だけではなく、将来枯渇や供給不安が懸念される元素も含む。

## 10. なぜ脆性破壊は危険か

延性破壊は大きな塑性変形を伴うため前兆が現れやすい。一方、脆性破壊は前兆が乏しく突然破壊に至るため、安全上より深刻である。

## 11. `クリープ` と `疲労`

材料は、明確な塑性変形開始条件に達していなくても、高温長時間使用で `クリープ` が進行し、繰返し応力で `疲労` が蓄積する。どちらも最終的には*微小き裂*の進展や*突然破壊*につながる。

## 12. `キーストーン種`

キーストーン種とは、生物量がそれほど大きくなくても、生態系全体に非常に大きな影響を与える種である。典型例は `ラッコ` で、ウニを捕食することで海藻林を維持する。

## 13. `PRTR` と `SDS`

`PRTR` は `Pollutant Release and Transfer Register` の略であり、事業者が指定化学物質の排出量・移動量を把握して国に届け出て、国が集計・公表する制度である。`SDS` は、化学物質の危険有害性、取扱い、保管、応急措置などを伝える安全データシート制度である。

補足記憶：

- `PRTR = 515種類`
- 排出量上位の代表物質は `トルエン`

## 14. なぜバイオマスは `カーボンニュートラル` と言われるのか

バイオマスは成長過程で大気中の `CO2` を光合成によって吸収し、その後燃焼などで放出する炭素が短期的な生物炭素循環に属するため、化石燃料より `カーボンニュートラル` と説明される。ただし、栽培、輸送、加工、土地利用変化に伴う排出は別途考慮が必要である。

## 15. 放射線防護の三原則

`時間`、`距離`、`遮へい` である。放射線源の近くにいる時間を短くし、距離をとり、間に適切な遮へい体を置く。

補足記憶：

- `α線 = ヘリウム原子核`
- `β線 = 電子`
- `γ線 / X線 = 電磁波`
- `中性子線` は `水` など水素を多く含む物質で弱めやすい
- 世界平均自然放射線は約 `2.4 mSv/年`
- 日本平均は約 `2.1 mSv/年`
- 体内天然核種として `カリウム40`、`炭素14`

## 15.5 `確定的影響` と `確率的影響`

`確定的影響` はしきい値があり、それを超えると影響が現れる。例として脱毛、白内障、組織障害がある。`確率的影響` は一般にしきい値がないと仮定し、線量が増えるほど発生確率が上がる。例としてがんや白血病がある。
