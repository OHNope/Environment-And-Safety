# 環境安全論 A/B 過去問の共通点と差分

## 分析範囲

この共通分析は、`source/problems/` にある 2022-2025 年の A/B 過去問を対象にする。2026 年小テストは各クラスの年度補正として使うため、ここでは混ぜない。

対象は次のように読む。

| 区分 | 使用した過去問 | 注意 |
|---|---|---|
| A | 2022 A 1Q、2022 A 2Q、2023 A 2Q、2024 A 1Q、2025 A 1Q、2025 A 2Q | 2024・2025 の 1Q A は構成が非常に近い |
| B | 2022 B 1Q、2023 B 1Q、2024 B 1Q、2024 B 2Q、2025 B 2Q | B は環境アセスメント、DT 核融合、材料安全が強い |

## 本当に A/B 共通で強い軸

| 共通軸 | A での出方 | B での出方 | 読み方 |
|---|---|---|---|
| エネルギーの形態と質 | 毎回級で出る。電気、熱、エクセルギーの説明が中心。 | 毎回級で出る。加えて熱機関・カルノー効率が混ざることがある。 | 最優先の共通テーマ。B だけでなく A 2Q でも熱機関が出たことはある。 |
| 廃プラスチック / 資源循環 | 2022・2023・A 2Q は MR / CR / TR、2024・2025 1Q は環境影響。 | MR / CR / TR と、MR・CR に必要なプラスチックの条件が定番。 | 共通だが、A 1Q 近年は `安定性 / 親油性 / 添加剤` へ寄る。 |
| 生命環境 | キーストーン種、エコロジカル・フットプリント、バイオマス。 | キーストーン種、バイオマス、PRTR との組み合わせ。 | 中頻度の共通候補。出る場所は Q6 付近が多い。 |
| 化学物質管理 | グリーンケミストリー、サステイナブルケミストリー、PRTR / SDS。 | PRTR / SDS、元素戦略、材料安全の正誤問題。 | 大分類は共通だが、細目は A/B でかなり違う。 |
| 持続可能性 | ティッピングポイント、正誤問題、個人の省エネ・省資源。 | 持続可能社会の課題と解決策の論述。 | 用語暗記より、論述の型を分けて準備する。 |

## A 側に寄せて読むべきもの

次は「A/B 共通」として扱うと混ざりやすい。A の分析では強く扱うが、B の定番とは分ける。

| テーマ | A での根拠 | B との違い |
|---|---|---|
| 放射線防護三原則 | A は `時間 / 距離 / 遮へい` と `確定的影響 / 確率的影響` が繰り返し出る。 | B の主軸は放射線防護ではなく、DT 核融合計算や核分裂・核融合の安全性比較。 |
| ティッピングポイント | A 1Q の Q2 で強い。 | B は同じ位置に環境アセスメントや持続可能社会論述が入る。 |
| 正誤問題 + 個人の行動 | A は 2024・2025 1Q で同型の正誤問題と省エネ・省資源の小論述が出る。 | B の正誤問題は元素戦略・材料安全系で、個人行動問題とは性質が違う。 |
| グリーンケミストリー本文説明 | A は目的、サステイナブルケミストリーで加わる概念、PRTR / SDS が直接問われる。 | B はグリーンケミストリー表題でも、実際は元素戦略や材料安全の正誤が中心。 |

## B 側に寄せて読むべきもの

次は B の固定軸であり、A の 1Q 対策へそのまま移すと過大評価になりやすい。

| テーマ | B での根拠 | A との違い |
|---|---|---|
| 環境アセスメント | 2023・2024 B 1Q、2024・2025 B 2Q で強い。再エネ事業の追加も B 2Q で反復。 | A 過去問では主軸ではない。A 2026 小テストにない限り、優先度は上げすぎない。 |
| DT 核融合計算 | B は DT 反応 1 回のエネルギー計算が定番。 | A 過去問の固定枠ではない。ただし 2026 A 小テストにゼロカーボンエネルギーがある場合は年度補正として別扱いする。 |
| 核分裂 vs 核融合の安全性 | B の原子力問題で出やすい。 | A の放射線問題は、防護三原則と健康影響の説明が中心。 |
| 元素戦略 / 材料安全 | B の正誤問題で強い。 | A の正誤問題は廃棄物、太陽光、化学物質リスク、バイオマス、温暖化、自然放射線など。 |
| 環境アセスメント + 再エネ | B 2Q で特に強い。 | A の持続可能性・気候変動対策とは近いが、制度論としては別テーマ。 |

## 混同しやすいポイント

1. `核融合` は、B では過去問固定テーマ、A では少なくとも過去問上の固定テーマではない。A で扱うなら、2026 A 小テストによる年度補正として扱う。
2. `放射線` は、A では防護と健康影響、B では核融合・原子力安全に寄る。同じ放射線系でも答案の型が違う。
3. `化学` は、A ではグリーンケミストリーと PRTR / SDS、B では元素戦略・材料安全へ寄る。
4. `持続可能性` は、A ではティッピングポイントや正誤問題、B では持続可能社会の課題と解決法の論述になりやすい。
5. `廃プラスチック` は共通だが、A 1Q 近年は環境影響、B は MR / CR / TR の制度・技術説明が強い。

## 2026 講義・小テストから見た差分

2026 年の A/B を比べると、次の分野はそのまま共通扱いにしない。

| 側 | 分野 | 扱い |
|---|---|---|
| B 寄り | 環境アセスメント | B 過去問・B 小テストで強い。A では優先度を上げすぎない。 |
| B 寄り | 物質・材料・元素戦略 | B の材料安全正誤問題につながる。A のグリーンケミストリー本文説明とは別物。 |
| B 寄り | 熱機関 / カルノー効率 | B では強い。A でも 2Q に出たことはあるが、A 1Q の本命はエネルギーの質。 |
| B 寄り | 放射線安全論の小テスト型知識 | B 2026 小テストでは強い。A 過去問では放射線防護三原則と影響分類が本命。 |
| A 寄り | グリーンケミストリー | A 過去問にも 2026 A 小テストにも対応するため、A では最優先級。 |
| A 寄り | ゼロカーボンエネルギー / 核融合 | A 過去問固定ではないが、2026 A 小テストで直接確認された。B 過去問の計算テンプレートを借りて対策する。 |
| A 寄り | 環境サステナビリティ | 2026 A 小テスト由来。A 過去問の固定大問ではないため、正誤・短答補強として扱う。 |
| A 寄り | 水文学系講義 | 現在の `source/2026_A_1Q_quizzes/` には対応小テストが未収録。資料を追加できるまではスコア対象外にし、扱う場合も小テスト補強として読む。 |

## 使い方

A の期末対策では、まず A 固有の型を優先する。具体的には、廃プラスチック環境影響、ティッピングポイント、放射線防護、エネルギーの質、グリーンケミストリー、PRTR / SDS、正誤問題、個人の省エネ・省資源である。

B の期末対策では、MR / CR / TR、環境アセスメント、エネルギーの質と熱機関、持続可能社会論述、DT 核融合計算、核分裂・核融合安全、元素戦略・材料安全を優先する。

共通分析は「大分類の地図」として使う。実際の予想順位は、必ず対象クラスの過去問と 2026 年小テストで補正する。
