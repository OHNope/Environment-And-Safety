## 1. 廃プラスチックの環境影響

プラスチックは化学的に安定で分解されにくいため、環境中に長期間残留する。さらに、紫外線や波浪、摩耗によって細かく砕かれ、マイクロプラスチックやナノプラスチックとなり、生物に摂取されるおそれがある。

また、プラスチックは親油性をもつため、疎水性の有機汚染物質を表面に吸着しやすい。これを生物が摂取すると、食物連鎖を通じて汚染物質が移動・蓄積する可能性がある。

さらに、プラスチックには可塑剤、難燃剤、安定剤などの添加剤が含まれる。これらが環境中で溶出すると、生物毒性や内分泌かく乱などのリスクを生む。

### 答案の型

廃プラスチック問題では、`安定性`、`親油性`、`添加剤` の 3 語を必ず明示する。最後に「長期残留、食物連鎖、添加剤溶出により、生態系や人の健康への影響が懸念される」とまとめると安定する。

## 2. `MR / CR / TR` と 2026 年小テスト補正

`MR` はマテリアルリサイクルであり、廃プラスチックを粉砕、洗浄、溶融、成形などの物理的処理によって再び材料として利用する方法である。`CR` はケミカルリサイクルであり、熱分解、ガス化、モノマー化などによって化学原料へ戻す方法である。`TR` はサーマルリサイクルであり、焼却や燃料化によって熱エネルギーを回収する方法である。

`MR / CR` には、単一素材に近く、汚れ・異物・劣化・添加剤混入の少ないプラスチックが必要である。混合物や汚れが多い場合は、材料としての再利用が難しくなり、`TR` に回りやすい。

2026 年小テストでは、3 種合計の有効利用率は `85%`、`TR` を除くと `25%` と整理されている。また、廃プラスチック `9.4百万トン` のうち `168万トン` が輸出され、主な輸出先は `中国` であった。国境を越える廃棄物移動の規制は `バーゼル条約` であり、2019 年に `廃プラスチック` が対象に追加された。

## 3. カーボンニュートラルへの科学技術の貢献

カーボンニュートラルに対して、科学技術は省エネルギー、再生可能エネルギー、蓄電池、水素、燃料電池、`CCS / CCUS`、カーボンリサイクル、材料の長寿命化、リサイクル高度化などで貢献できる。

ただし、使用時だけで排出量を見るのでは不十分である。原料採取、製造、輸送、使用、廃棄までを含むライフサイクル全体で、本当に温室効果ガス排出を減らしているか評価する必要がある。

## 4. ティッピングポイント

ティッピングポイントとは、地球環境システムがある閾値を超えると、変化が自己増幅し、急激・非線形・不可逆的に別の状態へ移る点である。

例として、永久凍土が融解するとメタンが放出され、温暖化がさらに進む。氷床が融解すると地表の反射率であるアルベドが低下し、太陽エネルギーを吸収しやすくなる。サンゴ礁では高水温により白化が進み、生態系が大きく変化する。森林枯死では炭素吸収力が低下し、温暖化をさらに進める可能性がある。

### 答案の型

まず定義を書く。次に具体例を 2 つ以上挙げ、それぞれ「なぜ自己増幅するか」を一言添える。単に例を列挙するだけでは弱い。

## 5. 持続可能性と気候変動の小テスト補正

`緩和` は、温室効果ガス排出を減らして気候変動そのものを抑える対策である。省エネルギー、再生可能エネルギー、電化、森林保全、`CCS` などが例である。

`適応` は、すでに起きる、または避けられない影響に備える対策である。洪水対策、熱中症対策、防災、農作物の品種改良などが例である。

`Our Common Future` は 1987 年に発表され、持続可能な開発の考え方を広めた。`MDGs` は 2000 年、8 目標であり、主に開発途上国の課題を対象にした。`SDGs` は 2015 年、17 目標であり、2030 年を目標年として先進国を含む全世界を対象にする。

`MPI` は多次元貧困指数であり、所得だけでなく教育、健康、生活水準など複数の欠乏から貧困を評価する。値は 0 から 1 の範囲をとり、大きいほど貧困の程度が深刻である。

## 6. 放射線防護の三原則

放射線防護の三原則は、`時間`、`距離`、`遮へい` である。被ばく時間を短くし、線源から距離をとり、線源との間に遮へい体を置くことで被ばくを低減する。

遮へい材料は放射線の種類によって異なる。`α線` はヘリウム原子核であり、紙や皮膚表面で止まりやすい。`β線` は電子であり、薄い金属板やプラスチックで弱められる。`γ線 / X線` は電磁波であり、鉛やコンクリートが有効である。`中性子線` は水やポリエチレンなど水素を多く含む物質で弱めやすい。

## 7. `確定的影響` と `確率的影響`

`確定的影響` は、しきい線量があり、それを超えると影響が現れる。線量が大きいほど症状の重症度が増す。例として、脱毛、白内障、皮膚障害、不妊などがある。

`確率的影響` は、一般にしきい値がないと仮定され、線量が大きいほど発生確率が増す。重症度ではなく、発生する確率が問題になる。例として、がん、白血病、遺伝的影響などがある。

自然放射線による一人当たりの世界平均被ばく量は約 `2.4 mSv/年` であり、`240 mSv/年` ではない。

## 8. エネルギーの形態と質の違い

例として、電気エネルギーと熱エネルギーを挙げる。電気エネルギーはモーターなどを通じて仕事へ変換しやすく、質が高い。一方、熱エネルギー、とくに周囲との温度差が小さい低温熱は、仕事へ変換しにくく、質が低い。

この違いは、同じエネルギー量のうち実際に利用可能な仕事量、すなわち `エクセルギー` の違いとして説明できる。エネルギー量が保存されても、すべてが有用な仕事に変わるわけではない。

### 正誤での注意

太陽光発電は熱機関ではなく、光を直接電気へ変換する。そのため、熱機関と同じ意味でカルノー効率の制約を受けるわけではない。

## 9. 水素・燃料電池・CCS

2026 年小テストでは、エネルギーの質に加えて水素や燃料電池も補正要素になる。

水素は、化石燃料、再生可能エネルギー、原子力など多様なエネルギー源から製造でき、貯蔵・運搬が可能なエネルギー媒体である。ただし、製造方法、輸送、貯蔵、コスト、安全管理が課題である。

燃料電池は、水素と酸素の化学反応から直接電気を取り出す装置である。発電時には主に水が生じるが、白金触媒やコスト低減が課題である。

`CCS` は二酸化炭素を分離・回収して貯留する技術であり、`CCUS` は回収した二酸化炭素を利用もする考え方である。

## 10. グリーンケミストリーとサステイナブルケミストリー

グリーンケミストリーの目的は、有害物質の使用や生成を減らす化学製品・プロセスを設計し、汚染を発生源で防ぐことである。後から処理するのではなく、最初から廃棄物や有害性を減らす設計を重視する。

サステイナブルケミストリーでは、これに加えて、化学物質や製品のライフサイクル全体を考える。原料選択、製造、使用、廃棄、資源循環、再生可能資源、エネルギー効率、社会・経済的持続可能性まで含めて評価する。

### 答案の型

グリーンケミストリーは「有害物質と廃棄物を発生源で減らす」。サステイナブルケミストリーは「ライフサイクル、資源循環、再生可能資源、エネルギー効率、社会・経済面まで広げる」。この 2 段で書けばよい。

## 11. `CFC / PFAS / PFOS / 新規化学物質リスク`

`CFC` は安定、不燃、低毒性で便利な物質として使われたが、その安定性のために成層圏まで到達し、オゾン層破壊を引き起こした。

`PFAS / PFOS` は撥水性、撥油性、耐熱性などで便利だが、難分解性で長期残留し、広域に拡散することが問題になる。

これらは、便利で安定な化学物質が、長期的には地球規模の環境リスクになる例である。そのため、化学物質は製造時だけでなく、使用、拡散、分解生成物、廃棄まで含めてライフサイクル全体で評価する必要がある。

`新規化学物質リスク` は、人間が作り出した新しい化学物質などによる未知の地球環境リスクを指す。惑星限界の文脈では、新しい物質による未知リスクとして扱われる。

補足記憶：

- `CAS REGISTRY`: 約 `300,000,000` 種
- 工業化学物質: 約 `40,000-60,000` 種

## 12. `PRTR` と `SDS`

`PRTR` 制度は、対象化学物質について、事業者が環境への排出量と廃棄物などとして事業所外へ移動した量を把握・届け出し、行政が集計・公表する制度である。

`SDS` 制度は、対象化学物質やそれを含む製品を譲渡・提供する際に、性状、危険有害性、取扱い、保管、応急措置などの情報を安全データシートで伝達する制度である。

どちらも、事業者による化学物質の自主的管理を促進し、環境保全上の支障を未然に防ぐための制度である。

補足記憶：

- `PRTR = 515種類`
- 排出量上位の代表物質は `トルエン`

## 13. 生命環境・バイオマス

`エコロジカル・フットプリント` は、人間活動を支えるために必要な生態系の需要量を土地面積などで示す指標である。

`キーストーン種` は、生物量や個体数は大きくなくても、生態系全体に大きな影響を与える種である。例として、ラッコはウニを捕食することで海藻林を維持する。

`フラッグシップ種` は、保全活動の象徴となる種である。2026 年小テストでは `トキ` が例になる。

バイオマスは、成長過程で大気中の `CO2` を光合成によって吸収し、その後燃焼などで放出する炭素が短期的な生物炭素循環に属するため、カーボンニュートラルと説明される。ただし、栽培、輸送、加工、土地利用変化に伴う排出は別途考慮する必要がある。

バイオリファイナリーは、生物資源から燃料、化学品、材料などを総合的に作る考え方である。原料の例は `生ゴミ / 活性汚泥 / 稲わら / 廃てんぷら油` である。

バイオマスプラスチックは炭素源がバイオマス由来のプラスチックである。生分解性プラスチックは微生物などで分解されるプラスチックであり、両者は同じではない。

## 14. 核融合とゼロカーボンエネルギー計算

`E = mc^2` は、質量欠損がエネルギーへ変換されることを表す。`DT` 核融合は、重水素 `D` と三重水素 `T` が反応し、ヘリウム、中性子、エネルギーを生じる反応である。

```
D + T → He + n + 17.6 MeV
```

放出エネルギーは `17.6 MeV = 1.76×10^7 eV` である。トリチウムは放射性水素同位体であり、内部被ばく管理が重要である。また、核融合炉ではトリチウム増殖のためにリチウム資源が重要になる。

核融合は連鎖反応ではなく、高温高密度条件が崩れると反応が止まりやすいため暴走しにくい。一方で、中性子照射、材料放射化、トリチウム管理は安全上の課題である。


## 16. 個人の省エネ・省資源

個人の行動を書く問題では、行動名だけで終わらせない。その行動が、電力消費、`CO2` 排出、資源消費、廃棄物のどれを減らすかを書く。

解答例：

冷暖房の設定温度を控えめにし、不要な照明を消すことで電力消費を減らす。公共交通、自転車、徒歩を利用することで自動車利用による燃料消費と `CO2` 排出を減らす。マイボトルや詰め替え製品を使い、食品ロスを減らし、分別を徹底することで、資源消費と廃棄物を減らす。

## 17. 2024・2025 年度 A 1Q 正誤問題

1. 廃棄物処理法によって、産業廃棄物の国を超えた移動は規制された。  
   `誤`。国境を越える有害廃棄物などの移動は、主に `バーゼル条約` と国内のバーゼル法の枠組みで規制される。

2. 太陽光発電は熱機関ではないので、カルノー効率を受けない。  
   `正`。太陽光発電は光を直接電気へ変換するため、熱機関と同じ意味でカルノー効率の制約を受けるわけではない。

3. 人工的な化学物質の与える影響は容易に予測できる。  
   `誤`。長期影響、複合影響、分解生成物、蓄積性、広域拡散などがあるため、環境影響を容易に予測することはできない。

4. 化学合成された乳酸を元に生成されたポリ乳酸はバイオマスプラスチックである。  
   `誤寄り`。植物由来乳酸から作るポリ乳酸ならバイオマスプラスチックだが、化学合成乳酸が化石資源由来ならバイオマスプラスチックではない。判断では炭素源を見る。

5. 二酸化炭素の排出量を 0 にして、しばらくすれば温暖化ははじめに戻る。  
   `誤`。排出を止めれば追加的な温暖化は抑えられるが、海洋の熱蓄積、長寿命の `CO2`、気候システムの遅れにより、気温がすぐ元に戻るわけではない。

6. 自然放射線による一人当たりの被ばく量の世界平均は約 240 ミリシーベルトである。  
   `誤`。世界平均の自然放射線は約 `2.4 mSv/年` であり、`240 mSv/年` ではない。
