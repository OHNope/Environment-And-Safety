# 2026 講義資料メモ からの重点補足

提供された 講義資料メモを読み、今年の予想に直接関係する部分だけを抽出した。

## 1. 環境アセスメント系講義資料メモ

この講義資料メモは `環境アセスメント` の講義資料である。

テキストから直接確認できる重要点は次のとおりである。

- `科学性` と `民主性`
- 目的: 未然防止、環境配慮、コミュニケーション、社会的受容性
- 制度の流れ:
  - `1997` 制定
  - `1999` 施行
  - `2011` 風力発電を対象化
  - `2020` 太陽光発電を対象化
  - `2024` 洋上風力 central 方式
- 対象事業:
  - 新幹線、ダム、風力、太陽光、最終処分場など
- `スクリーニング`
- `リニア中央新幹線`
  - 騒音・振動
  - 地下水 / 河川流量
  - 大量のエネルギー消費
  - 発生土処理、景観影響

### 予想への影響

`Q2` の確度はさらに上がった。出題形式は次の 4 種類のいずれかになりやすい。

1. 科学性と民主性
2. 再生可能エネルギー事業が環境アセスメント対象になった理由
3. スクリーニングの問題
4. リニア中央新幹線の具体的影響

## 2. 原子力・放射線系講義資料メモ

この講義資料メモは `原子力と環境安全` の講義資料である。

テキストから直接確認できる重要点は次のとおりである。

- 放射線の種類:
  - `α = ヘリウム原子核`
  - `β = 電子`
  - `中性子線 = 中性子`
  - `γ線 = 原子核から飛び出してくるもの`
- 放射線防護三原則:
  - `遮へい`
  - `距離`
  - `時間`
- 自然放射線の由来:
  - 宇宙線
  - 大地
  - `ラドン`
- 体内天然核種:
  - `カリウム40`
  - `炭素14`
- 影響分類:
  - `確定的影響`
  - `確率的影響`
- 数値:
  - 世界平均自然放射線は約 `2.4 mSv/年`
  - 日本平均は約 `2.1 mSv/年`

### 予想への影響

今年 B クラス final に放射線が入る場合、単なる常識問題ではなく、次のような講義内の整理問題になる可能性が高い。

1. 放射線の種類と遮へい
2. 放射線防護三原則
3. 確定的影響 / 確率的影響
4. 自然放射線の由来と基本数値

## 3. 生命理工系講義資料メモ

この講義資料メモ には、生命理工系の有用な部分が含まれている。

テキストから直接確認できる重要点は次のとおりである。

- `エコロジカル・フットプリント`
  - 講義資料上で `ここだ！` と強調されている。
- `キーストーン種`
  - 定義ページでも `ここだ！` と強調されている。

  - `Pollutant Release and Transfer Register`

  - `生分解性プラスチック`

### 予想への影響

`Q6` の事実型対策では、次を優先して押さえるべきである。

1. キーストーン種
2. エコロジカル・フットプリント
3. PRTR
4. バイオプラスチック / 生分解性プラスチック

## 4. 現在の復習に最も関係する結論

今年追加で暗記すべき優先項目は次のとおりである。

1. `環境アセスメント` の具体的制度
2. `PRTR = 515 / トルエン`

## 5. 資源・廃棄物系講義資料メモ 第 1 回講義補足

この講義資料メモ が `資源と廃棄物` の原講義資料であり、予想上の価値は高い。

テキストから直接確認できる重要点は次のとおりである。

- 一般廃棄物:
  - `1人1日 約1kg`
  - 最終処分場の残余年数 `20.5年`
  - `東京ゴミ戦争`
- プラスチック問題の三要素:
  - `高い安定性`
  - `親油性`
  - `添加剤`
- リスク:
  - マイクロプラスチック
  - ナノプラスチック
  - `POPs` の吸着と運搬
- リサイクル:
  - `MR / CR / TR`
  - 日本では焼却技術が発達している
  - 過去には大量輸出もあった
- 国際規制:
  - `バーゼル条約`
  - `2019` 年改正で `汚れたプラスチックごみ` が対象に入った
- `3R`
  - `2000年 循環型社会形成推進基本法` と明確に対応する
- 発展的対策:
  - `バイオプラスチック`
  - `CCS / CCUS`
  - `CO2` 資源化
- 学内管理:
  - `PRTR`
  - `IASO`

### 予想への影響

この講義資料メモ により、次の 3 種類の問題の重みが上がる。

1. `Q1` の制度・現実背景問題
   定義だけでなく、日本で `TR` が多い理由や、プラスチック問題がなぜ深刻かを問う可能性がある。

2. `正誤問題` における条約問題
   `バーゼル条約` はほぼ必須暗記事項である。

3. `Q6` における管理制度問題
   `PRTR` と `IASO` は第 1 回講義にも出ているため、生命理工系講義だけの内容ではない。
