# B クラス期末試験の出題方向と頻度分析

## 1. サンプル範囲

本ファイルでは B クラスのみを扱う。

- `2022_B_1Q`
- `2023_B_1Q`
- `2024_B_1Q`
- `2024_B_2Q`
- `2025_B_2Q`

さらに、`2026_B_1Q_quizzes` を用いて今年の方向を補正する。

補足:

- `2024_B_2Q` と `2025_B_2Q` は 1Q ではないが、問題構成が B クラスの固定テンプレートにかなり近いため、B クラスの出題習慣を読む材料として使える。
- `2022_B_1Q` には再現が不完全な部分がある。該当箇所は `probable` として扱う。

## 2. B クラス最大の特徴: 試験構成が非常に安定している

2023 年から 2025 年にかけて、B クラスはほぼ次の固定構成になっている。

1. `Q1` 資源と廃棄物: MR / CR / TR
2. `Q2` 環境アセスメント
3. `Q3` エネルギー形態とエネルギーの質
4. `Q4` 持続可能社会に関する総合論述
5. `Q5` 核融合計算 + 関連する追問
6. `Q6` 生命環境 / 化学物質 / バイオ技術系
7. `Q7` 元素戦略 / 材料安全の正誤問題

これは単なる類似ではなく、すでにテンプレートに近い。

### 3.3 2026 年 講義資料メモによる補正

ここでは頻度を再計算するのではなく、講義資料で実際に強調されている点を確認する。

1. `環境アセスメント` 講義資料メモは Q2 の高い重要度を直接裏づける。
   - `科学性 / 民主性`
   - `1997制定・1999施行`
   - `2011 風力発電を対象化`
   - `2020 太陽光発電を対象化`
   - `スクリーニング`
   - `リニア中央新幹線`

2. `小原` 講義資料メモは、放射線が単なる quiz レベルの常識ではないことを示す。
   - 放射線の種類と遮へい
   - `時間・距離・遮へい`
   - `確定的影響` と `確率的影響`
   - 世界平均自然放射線 `2.4 mSv/年`
   - 日本平均約 `2.1 mSv/年`
   - `ラドン`, `カリウム40`, `炭素14`

3. `1Q_compressed` 講義資料メモは Q6 の生命理工系テーマを補強する。
   - `エコロジカル・フットプリント`
   - `キーストーン種`
   - `PRTR = 515種類`
   - 年間排出量上位物質の第 1 位は `トルエン`
   - `バイオプラスチック`
   - `生分解性プラスチック`

4. `2026_B_1Q` 第 1 回 quiz は Q1 を直接補強する。
   - `MR / CR / TR` の定義
   - 日本の廃プラスチック有効利用率 `85%`
   - `TR` を除いた率 `25%`
   - 廃プラスチック輸出量 `168万トン / 9.4百万トン`
   - 最大輸出先 `中国`
   - `バーゼル条約`
   - 2019 年に対象追加された `廃プラスチック`
   - カーボンニュートラルに対して科学がどう貢献できるか

## 4. B クラスで今年最も出やすい方向

### S ランク: ほぼ必ず準備すべきもの

1. `Q1 プラスチックリサイクル`
   - B クラスでは全回出題されている。
   - 2026 年 B 第 1 回 quiz でも直接出題されたため、過去問頻度だけでなく今年度 quiz からも強く支持された。
   - 基本要求は `MR / CR / TR` の定義と、`MR / CR` に必要なプラスチックの純度・成分条件である。
   - 今年の quiz に合わせるなら、数値は `85%`、`25%`、`9.4百万トン`、`168万トン`、語句は `中国`、`バーゼル条約`、`廃プラスチック` を暗記する。
   - 第 1 回講義資料メモ 末尾まとめもこの重要性を支持している。教員はリサイクル定義だけでなく、`一般廃棄物の現状`、プラスチック廃棄物の地球規模の蓄積、マイクロプラスチック / ナノプラスチックのリスク、`3R`、`バイオプラスチック`、`カーボンリサイクル` まで扱っている。
   - 原 講義資料メモを踏まえると、`バーゼル条約` と「日本で `TR` が多くなりやすい理由」も追加で重要である。したがって Q1 は単なる概念問題ではなく、制度背景や現実的理由を含む問題になる可能性がある。

2. `Q3 エネルギー形態とエネルギーの質`
   - これも全回出題である。
   - 今年の quiz では、熱機関、エクセルギー、カルノー効率が強化されている。

3. `Q4 持続可能社会に関する総合論述`
   - 細かい知識を並べる問題ではなく、科目の主軸を組み立てて解答化できるかが問われる。

4. `Q5 DT 核融合計算`
   - 5 年分すべてに出ている。
   - 数値はほぼ固定で、標準的な答えは `1.8×10^7 eV` である。

5. `Q7 元素戦略 / 材料安全の正誤問題`
   - B クラスを代表する問題である。
   - 2026 年 quiz でも、過去に使われた選択肢に近い内容が再び出ている。

### A ランク: 高確率

1. `Q2 環境アセスメント`
   - 5 / 5 ではないが、2023 年以降は非常に安定している。
   - 今年の quiz 支持も全テーマ中で最も強い。
   - 出題される場合、次のどれかに寄る可能性が高い。
     - 科学性と民主性
     - 再生可能エネルギー事業が環境アセスメント対象になった背景
     - スクリーニングの問題
     - リニア中央新幹線の事例

2. `Q6 生命環境系`
   - 過去にはキーストーン種、MPI、バイオマスのカーボンニュートラルが出ている。
   - 今年の quiz では、キーストーン種、PRTR、エコロジカル・フットプリント、バイオリファイナリー、発がん性が広く扱われた。
   - 講義資料メモを見ると、`PRTR` と `エコロジカル・フットプリント` は周辺知識ではなく、講義資料上でも明確に強調されている。

### B ランク: 今年明らかに重要度が上がったもの

1. `熱機関 / カルノー効率`
   - 2024 B2Q、2025 B2Q で連続して出ている。
   - 2026 年にも熱機関 quiz がある。
   - 今年は Q3 に組み込まれ、「エネルギーの質 + 最大熱効率」の二段構成になる可能性が高い。

2. `放射線防護 / 自然放射線`
   - 歴史的には B クラスの主軸ではない。
   - ただし今年は独立した quiz があり、少なくとも記憶レベルでの準備が必要である。
   - `小原` 講義資料メモを見ると、実際に強調されているのは、放射線の種類、三原則、確定的影響 / 確率的影響、自然放射線の由来と数値である。
   - Q5 の第二小問、または Q6 に入る可能性がある。

## 5. 2026 1Q B の設問別予想

### 予想 Q1

最も可能性が高いのは次の内容である。

- `MR / CR / TR` の定義
- 日本の廃プラスチックリサイクル率、有効利用率、`TR` を除いた比率
- 廃プラスチック輸出先としての `中国` と、輸入禁止後の影響
- `バーゼル条約` と 2019 年の `廃プラスチック` 対象追加
- なぜ `TR` が `MR / CR` より多くなりやすいか
- `MR / CR` にはどのようなプラスチックが必要か

第 1 回 quiz の第 2 問に寄せる場合、次のような論述が追加される可能性がある。

- カーボンニュートラル実現に向けて、科学がどう貢献できるか
- リサイクル、材料開発、CCUS、カーボンリサイクル、省エネルギーをどう組み合わせるか

第 1 回講義資料メモ により近い出題になる場合は、次のいずれかも追加される可能性がある。

- プラスチック廃棄物が地球規模問題になる理由
- マイクロプラスチック / ナノプラスチックの生態系・健康影響
- `3R` 以外の解決方向: `バイオマスプラスチック`、`生分解性プラスチック`、`CCUS / カーボンリサイクル`

したがって、Q1 は「必出の定義問題」から「数値・制度・現状・対策まで含みうる高確率大問」として準備すべきである。

### 予想 Q2

今年は次の方向が特に可能性が高い。

- 環境アセスメントの科学性と民主性
- 再生可能エネルギー事業が環境アセスメント対象になった背景と課題
- スクリーニングの問題

2026 年 quiz により寄せる場合、`リニア中央新幹線` が具体例として問われる可能性もある。

### 予想 Q3

かなり高い確率で、次の二段構成になる。

- エネルギー形態を 2 つ挙げ、その質の違いを説明する。
- `1000℃`、`25℃` の熱機関について最大効率を計算する。

### 予想 Q4

引き続き、開放型の総合論述になる可能性が高い。

- 持続可能社会に向けた課題
- 解決策
- 技術、制度、資源、社会的受容性をつなげた解答構成

### 予想 Q5

最も安定しているのは次である。

- `Q5(i)` DT 核融合 1 回あたりのエネルギー計算

第二小問の候補は次の順で考える。

1. 核分裂と核融合の安全性比較
2. 放射線防護 / 自然放射線
3. MPI

優先度は `1 > 2 > 3` と見る。

### 予想 Q6

候補は最も複雑である。確率順に並べると次のようになる。

1. キーストーン種 / 生物多様性
2. PRTR / SDS または PRTR 関連管理
3. エコロジカル・フットプリント
4. バイオマスがカーボンニュートラルとされる理由
5. バイオリファイナリー / バイオプラスチック

### 予想 Q7

引き続き、5 問前後の正誤判断になる可能性が高い。素材は次が中心である。

- 元素戦略
- RoHS と将来枯渇が懸念される元素
- クリープ / 疲労
- 延性破壊 / 脆性破壊
- 採掘困難な資源も回収対象として考える理由

## 6. 現段階で追加講義資料 を見るなら

予想精度をさらに上げるなら、優先度が高い 講義資料 は次である。

1. `環境アセスメント`
   - Q2 の具体的な問われ方に直接関わる。
2. `生命理工学院の講義`
   - Q6 がキーストーン種、PRTR、エコロジカル・フットプリント、バイオマス、バイオリファイナリーのどれに寄るかに関わる。
3. `放射線安全`
   - Q5 第二小問、または Q6 が放射線に差し替わるかどうかに関わる。

優先順位は `環境アセスメント > 生命理工 > 放射線安全` と考える。

補足: すでに提供された `資源と廃棄物` 第 1 回まとめに加えて、2026 年 B 第 1 回 quiz が Q1 の直接証拠になった。現時点では、Q1 は `MR / CR / TR` の定義だけでなく、プラスチック廃棄物の数値、輸出先、`バーゼル条約`、現状、リスク、制度、対策まで含めて準備するべきである。
