# A クラス期末試験の出題方向と頻度分析

## 1. 分析範囲

本ファイルでは `2026 1Q 環境安全論 A` を対象にする。

使用資料は次の通りである。

- `source/problems/` 内の A クラス過去問。
- `source/2026_A_1Q_quizzes/` 内の 2026 A quiz。
- [共通傾向分析](../common/overall_trend_analysis.md)。

頻度集計は [scripts/generate_analysis.py](scripts/generate_analysis.py) で生成し、結果は [build/topic_frequency.md](build/topic_frequency.md) と [build/prediction_scoreboard.tsv](build/prediction_scoreboard.tsv) に置く。

数値スコアでは、過去問または 2026 A 小テストで確認できるものを優先する。講義資料メモだけの根拠は、補足説明には使うが、単独ではランクを上げない。

## 2. A クラス最大の特徴

A クラスは、B クラスほど工学計算テンプレートには寄らない。一方で、2024・2025 の 1Q A は非常に近い構成になっている。

典型構成は次である。

1. `Q1` 廃プラスチック問題
2. `Q2` ティッピングポイント
3. `Q3` 放射線防護三原則 + 確定的影響 / 確率的影響
4. `Q4` エネルギー形態とエネルギーの質
5. `Q5` グリーンケミストリー / サステイナブルケミストリー
6. `Q6` PRTR / SDS または生命環境系
7. `Q7` 総合正誤問題
8. `Q8` 個人の省エネルギー・省資源実践

このため、A クラスでは「固定テンプレート + その年の quiz 補正」で読むのが最も実用的である。

### 3.3 2026 年補正後の重要度

最新の自動集計は [build/topic_frequency.md](build/topic_frequency.md) を参照する。読み取りは次の三層に分ける。

| 層 | テーマ | 根拠 |
|---|---|---|
| 最優先 | エネルギーの質、放射線防護、放射線影響、総合正誤、個人の行動、廃プラスチック、グリーンケミストリー、キーストーン種 | A 過去問で反復し、テーマによっては 2026 小テストも対応 |
| A 級 | ティッピングポイント、核融合 / 重水素・三重水素 | ティッピングポイントは A 過去問反復。核融合は A 過去問固定ではないが、2026 A 小テストで計算まで直接確認され、B 過去問にも完成した出題型がある |
| 小テスト補強 | 持続可能性 / SDGs、CFC / PFAS、水素 / CCS / 燃料電池、バイオリファイナリー等 | 2026 A 小テストには出たが、A 過去問の固定大問としては弱い。正誤問題・部分点・差し替え候補として準備する |

したがって、2026 A の読み方は「A 1Q の固定テンプレートを崩さず、小テストで出た A 独自講義を補強する」である。小テストだけで支持される分野は、項目数が多くても過去問反復テーマと同列には置かない。

## 4. 2026 quiz と 講義資料メモによる補正

### 4.1 廃プラスチック / 資源・廃プラスチック系講義資料メモ

2026 A 第 1 回 quiz は、Q1 の方向をかなり強く補正する。

重要語句:

- `マテリアルリサイクル`
- `ケミカルリサイクル`
- `サーマルリサイクル`
- 3 種合計の利用率 `85%`
- TR を除いた率 `25%`
- 廃プラスチック `9.4百万トン` 中 `168万トン` を輸出
- 主な輸出先 `中国`
- `バーゼル条約`
- 2019 年に追加された `廃プラスチック`
- カーボンニュートラルに対する科学の貢献

したがって Q1 は、2024・2025 型の `安定性 / 親油性 / 添加剤` に加えて、MR / CR / TR と制度・数値を準備する必要がある。

### 4.2 エネルギー / 水素 / CCS

2026 A energy quiz は、エネルギーの質に加え、次を直接扱っている。

- エネルギー保存
- エネルギーの質
- 熱移動の方向性
- パリ協定
- `3E+S`
- CCS
- 水素の製造・貯蔵・運搬
- 固体高分子形燃料電池
- 白金触媒
- コスト低減

Q4 は従来通り `エネルギー形態と質` が本命である。ただし、正誤問題または総合論述では水素・CCS・燃料電池が混ざる可能性がある。

### 4.3 持続可能性 / 持続可能性系講義資料メモ

持続可能性系講義資料メモ と sustainability quiz は次を補強する。

- `Our Common Future`
- MDGs と SDGs の時系列
- 緩和と適応
- CO2 排出削減後も気温がすぐ戻らないこと
- 法律・政策が科学だけで決まるわけではないこと
- 世界人口、森林、地球温暖化

Q2 は引き続きティッピングポイントが本命である。sustainability quiz の事実問題は、Q2 の丸ごと差し替え本命というより、正誤問題または短答補強として警戒する。

### 4.4 グリーンケミストリー / グリーンケミストリー系講義資料メモ

グリーンケミストリー系講義資料メモ とグリーンケミストリー小テストは完全に対応している。

重要論点:

- グリーンケミストリーの基本思想
- 化学物質のライフサイクル
- CFC が便利さと安定性のために普及し、後に地球規模問題となったこと
- PFAS / PFOS の難分解性・広域汚染
- Novel entities

Q5 は従来の「グリーンケミストリーの目的」だけでは不十分である。CFC / PFAS を例にして、ライフサイクルと未知リスクまで説明できる必要がある。

### 4.5 生命理工 quiz

生命理工 quiz は Q6 候補を広げる。

重要論点:

- エコロジカル・フットプリント
- 生物多様性
- キーストーン種
- フラッグシップ種
- PRTR `515` 物質
- トルエン
- IARC Group 1
- バイオマスのカーボンニュートラル
- バイオリファイナリー
- バイオマスプラスチック

Q6 は PRTR / SDS が過去問上は強いが、2026 quiz だけを見ると生命環境系への差し替えも十分あり得る。

### 4.6 ゼロカーボンエネルギー / 核融合

2026 年講義資料メモ は MDGs / SDGs、エネルギー、核融合炉、重水素、三重水素を扱う。さらに、新たに確認されたゼロカーボンエネルギー小テストでは、SDGs / MDGs、MPI、質量エネルギー等価、太陽の質量欠損、DT 反応、核融合炉のリチウム消費量、海水中リチウム資源が直接問われている。

これにより、核融合は「A 過去問固定テーマ」ではなく「2026 A 小テスト直撃 + B 過去問に既存テンプレートがある変化球」として扱う。過去問上は放射線防護三原則と確定的影響 / 確率的影響が依然として最本命であり、核融合は Q4 のエネルギー論述、Q7 の正誤問題、または追加計算として備える位置づけである。

特に準備すべきもの:

- `E = mc^2`。
- 太陽の質量欠損から残り年数を求める計算。
- DT 反応の放出エネルギー `17.6 MeV`。
- 100 g の質量欠損で水を加熱する概算。
- 1 GW 核融合炉の年間リチウム消費量。
- 海水中リチウム資源で 1000 基の核融合炉を何年動かせるか。

## 5. 2026 A の設問別予想

### 予想 Q1

最有力は次の複合型である。

- プラスチックの環境影響を `安定性 / 親油性 / 添加剤` で説明する。
- `MR / CR / TR` の定義と違いを説明する。
- `85%`、`25%`、`中国`、`バーゼル条約`、`廃プラスチック` を答える。
- カーボンニュートラルに対して科学がどう貢献できるか述べる。

### 予想 Q2

本命は `ティッピングポイント` である。

例として、永久凍土融解、氷床融解、サンゴ礁白化、森林枯死を使えるようにする。2026 補正として、SDGs、Our Common Future、緩和と適応、CO2 削減後の気温持続も正誤問題で警戒する。

### 予想 Q3

最も安定しているのは放射線である。

- 放射線防護三原則: `時間 / 距離 / 遮へい`
- 確定的影響: しきい値あり、重症度が線量で増える
- 確率的影響: 確率が線量で増える、がん・遺伝的影響

### 予想 Q4

`エネルギー形態とエネルギーの質` が本命である。

電気エネルギーと低温熱を比較し、利用可能な仕事量、exergy、熱移動の方向性に触れる。2026 quiz により、CCS、水素、燃料電池に加えて、`E=mc^2`、太陽の質量欠損、DT 核融合、リチウム資源計算が混ざる可能性がある。ただし、A の本命答案はあくまでエネルギーの質であり、B 型の熱機関・核融合計算へ過度に寄せすぎない。

### 予想 Q5

`グリーンケミストリー / サステイナブルケミストリー` が本命である。

今年は次も準備する。

- ライフサイクル
- CFC
- PFAS / PFOS
- Novel entities
- 「便利で安定な物質」が後から地球規模リスクになる構造

### 予想 Q6

候補順は次である。

1. PRTR / SDS
2. エコロジカル・フットプリント
3. キーストーン種
4. バイオマスのカーボンニュートラル
5. バイオリファイナリー / バイオマスプラスチック

### 予想 Q7

正誤問題は高確率で出る。特に次を暗記する。

- 廃プラスチックの越境移動規制はパリ協定ではない。
- 太陽光発電は熱機関ではないので、同じ意味でカルノー効率制限を受けない。
- 人工化学物質の環境影響は容易には予測できない。
- バイオマスプラスチックかどうかは炭素源を見る。
- CO2 排出を減らしても気温はすぐ元に戻らない。
- 世界平均自然放射線量は約 `2.4 mSv/年` であり、`240 mSv/年` ではない。
- SDGs は 17 目標、MDGs は 8 目標である。
- `MPI` は多次元貧困指数であり、0 から 1 の値をとり、大きいほど貧困が深刻である。
- DT 反応の代表的な放出エネルギーは約 `17.6 MeV` である。
- 核融合炉ではトリチウム増殖のためにリチウム資源が重要になる。

### 予想 Q8

個人の省エネルギー・省資源実践が出る可能性が高い。単なる行動列挙ではなく、なぜ省エネルギー・省資源につながるかを書く。

## 6. 結論

A クラスは、従来は B クラスのような核融合計算・環境アセスメント固定型ではなかった。2026 A では、2024・2025 の安定テンプレートを基本にしつつ、quiz により次が上乗せされている。

- 廃プラスチックの数値・制度・バーゼル条約
- CFC / PFAS / Novel entities
- SDGs / Our Common Future / 緩和と適応
- 水素 / CCS / 燃料電池
- ゼロカーボンエネルギー、DT 核融合、質量欠損、リチウム資源
- PRTR、エコロジカル・フットプリント、バイオマス系

最優先は `放射線`、`エネルギーの質`、`廃プラスチック`、`グリーンケミストリー`、`ティッピングポイント` である。核融合関連は A 過去問固定ではないが、今回のゼロカーボンエネルギー小テストにより A 級の変化球へ上がったため、少なくとも `DT 反応 17.6 MeV`、`E=mc^2`、`リチウム消費量`、`海水中リチウム資源` は計算手順まで確認する。一方、持続可能性系の単独大問化は過去問支援が弱いため、正誤・短答補強として扱う。
