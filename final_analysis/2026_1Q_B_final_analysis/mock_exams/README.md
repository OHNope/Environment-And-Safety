# B クラス予想模擬試験

3 セットを用途別に分けている。

| セット | 用途 | 問題 | 解答 |
|---|---|---|---|
| Set 1 | 本命テンプレート | `set1_core_questions.md` | `set1_core_answers.md` |
| Set 2 | 2026 quiz 寄せ | `set2_quiz_focused_questions.md` | `set2_quiz_focused_answers.md` |
| Set 3 | 変化球対策 | `set3_variant_questions.md` | `set3_variant_answers.md` |

## 使い方

1. まず `set1_core_questions.md` を時間を決めて解く。
2. `set1_core_answers.md` で、解答の型を確認する。
3. Set 2 で 2026 年 quiz の語句・数値を確認する。
4. Set 3 で、問われ方が変わった場合の説明力を確認する。

答案を見ながら覚えるだけではなく、各セットの問1、問2、問3、問5は一度手書きで答案化する。
